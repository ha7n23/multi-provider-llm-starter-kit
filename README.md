# Multi-Provider LLM Starter Kit

A production-style Python starter kit for building LLM applications with multiple providers, clean architecture, structured outputs, streaming, tests, and later FastAPI deployment.

## Current Phase

Phase 1A: project foundation with mock LLM provider.

## Current Features

- Clean `src/` project structure
- Environment-based configuration
- Logging setup
- Custom exceptions
- Provider interface
- Mock LLM client
- Streaming interface
- Banking complaint triage prompt
- Structured JSON parsing
- Pydantic validation
- Unit tests with pytest
